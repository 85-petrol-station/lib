# 快速开始指南 - 喇叭按钮

## 🎯 核心功能

在四级/六级的**拼写**和**识词**模块中添加了喇叭🔊按钮，可以：
1. **点击播放** - 立即播放单词的英文读音
2. **上传图片** - 用自己的PNG图片替换喇叭图标

---

## 🔊 如何使用

### 方式1：播放读音（推荐）
```
1. 打开任意识词或拼写模块
2. 看到单词后，直接点击喇叭按钮🔊
3. 完成！系统自动播放英文读音
```

### 方式2：上传自定义图标
```
1. 右键点击喇叭按钮（手机长按）
2. 选择"打开文件"对话框
3. 选择你的PNG图片（建议大小18x18px）
4. 完成！所有喇叭按钮都更新为新图片
```

## 📝 技术说明

### 播放原理
- 使用**Web Speech API**
- 自动识别当前单词
- 美式英文发音
- 支持多次点击（自动打断前一个）

### 图片存储
- 格式：**PNG**
- 位置：浏览器**localStorage**
- 持久化：刷新页面后保留
- 大小：建议18x18像素

### 默认图标
- 无图片时显示：**🔊**表情
- 加载失败时：自动降级到表情
- 完全兼容所有浏览器

## ❓ 常见问题

**Q: 为什么没有声音？**
```
A: 1. 检查浏览器音量是否为0
   2. 确认浏览器允许Web Speech API
   3. 尝试刷新页面重新加载
```

**Q: 如何恢复默认喇叭图标？**
```
A: 方法1（简单）：清除浏览器缓存
   方法2（精准）：浏览器开发者工具 → Application → 
                 localStorage → 删除 speakerImageData
```

**Q: 上传的图片不显示？**
```
A: 1. 确保上传的是PNG格式
   2. 图片文件不要太小（建议18x18px以上）
   3. 尝试刷新页面
```

**Q: 多个设备能同步吗？**
```
A: 不能。图片保存在本机浏览器，不同设备/浏览器需分别上传
```

## 🛠️ 高级用法

### 快速切换图标
```
1. 直接右键点击喇叭按钮
2. 选择新的PNG图片
3. 所有4个喇叭按钮立即更新
```

### 恢复默认状态
```
浏览器开发者工具：
→ F12
→ 应用程序 (Application)
→ 本地存储 (localStorage)
→ 当前网址
→ 查找 speakerImageData
→ 右键删除
→ 刷新页面
```

### 检查状态
```
浏览器控制台输入：
localStorage.getItem('speakerImageData')

如果返回值存在，说明已上传图片
如果返回 null，说明使用默认图标
```
