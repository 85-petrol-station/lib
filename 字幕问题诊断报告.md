# 🎬 Yibo News 字幕加载问题诊断报告

## 问题症状
- ❌ Yibo News (BBC News) 的字幕无法加载
- 显示"暂无字幕"提示
- 但字幕文件 `audio_1.vtt` 确实存在于项目目录中

---

## 🔍 根本原因分析

### 发现的问题：

#### 1️⃣ **Fetch 跨域 / 文件访问问题**
在 `index copy_0520_副本2.html` 中，`loadNewsSubtitle()` 函数使用 `fetch()` 加载 VTT 文件：
```javascript
async function loadNewsSubtitle(audioSrc) {
  const subSrc = audioSrc.replace('.mp3', '.vtt').replace('.m4a', '.vtt');
  const res = await fetch(subSrc);  // ← 问题在这里
  if (!res.ok) throw '无字幕文件';  // 失败时直接抛出异常
  // ...
}
```

**为什么会失败：**
- 如果通过 `file://` 协议打开 HTML，浏览器会阻止 `fetch()` 访问同目录文件（安全限制）
- 如果通过 HTTP 访问但服务器配置有误，可能导致 CORS 错误
- 网络请求失败时没有详细的错误日志

#### 2️⃣ **缺少错误诊断机制**
```javascript
catch(e){
  document.getElementById('newsSubtitleList').innerHTML = `
    <div style="text-align:center;padding:10px;color:var(--color-text-secondary);">
      暂无字幕
    </div>`;
  newsSubtitles = [];
}
```
- 错误被吞掉了，无法知道具体原因
- 没有在 `console.error()` 中记录错误信息

#### 3️⃣ **可能的数据问题**
在 `ensureDefaultBbcNews()` 中：
```javascript
sub: 'audio_1.vtt',  // 只是文件名，没有完整路径
audio: 'audio_1.mp3',
```

### 方案 2️⃣：改进 JavaScript 代码（增强错误处理）

修改 `index copy_0520_副本2.html` 中的 `loadNewsSubtitle()` 函数：

**替换现有代码：**
```javascript
async function loadNewsSubtitle(audioSrc) {
  const subSrc = audioSrc.replace('.mp3', '.vtt').replace('.m4a', '.vtt');
  try {
    console.log('[字幕加载] 尝试加载:', subSrc);
    const res = await fetch(subSrc);
    
    if (!res.ok) {
      console.error(`[字幕加载] HTTP ${res.status}: ${res.statusText}`);
      throw `HTTP ${res.status}`;
    }
    
    const text = await res.text();
    console.log('[字幕加载] 成功读取', text.length, '字符');
    
    newsSubtitles = parseVtt(text);
    console.log('[字幕加载] 解析成功，共', newsSubtitles.length, '条字幕');
    
    renderNewsSubtitles();
  } catch (e) {
    console.error('[字幕加载] 错误详情:', e);
    console.warn('[字幕加载] 字幕文件:', subSrc);
    console.info('[字幕加载] 建议: 检查浏览器控制台了解详细错误');
    
    document.getElementById('newsSubtitleList').innerHTML = `
      <div style="text-align:center;padding:10px;color:var(--color-text-secondary);">
        ⚠️ 暂无字幕 - 请检查浏览器控制台(F12)获取详细信息
      </div>`;
    newsSubtitles = [];
  }
}
```
