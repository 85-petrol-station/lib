# Yibo News VTT 字幕问题 - 完整解决方案

## 📋 快速摘要

**问题**：Yibo News 模块显示"暂无字幕"

**原因**：浏览器通过 `file://` 协议打开 HTML 时，`fetch()` 被安全限制阻止

**字幕命名规则**：VTT 字幕必须与音频文件同名
```
audio_1.mp3  →  audio_1.vtt  ✓ 正确
exploring1_1.m4a  →  exploring1_1.vtt  ✓ 正确
```

## 🆘 常见问题

### Q1：为什么用 `file://` 打开 HTML 会显示"暂无字幕"？
**A**：浏览器安全限制。解决：使用 Web 服务器而不是直接打开文件。

### Q2：如何知道字幕是否成功加载了？
**A**：
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查找 `[字幕加载]` 开头的日志信息

### Q3：字幕文件一定要放在项目根目录吗？
**A**：根据当前代码，是的。所有 MP3 和 VTT 文件应该在同一目录。

### Q4：是否支持子目录中的字幕文件？
**A**：可以，但需要修改 UI 中的文件路径。例如：
```
音频：audio/news_title.mp3
字幕：audio/news_title.vtt  ← 必须包含完整路径
```
