## 功能特性

### 播放读音
- 使用Web Speech API实现英文读音
- 点击按钮立即播放当前单词发音
- 采用美式英文发音（en-US）

### 自定义图标
- 支持上传PNG格式图片
- 通过右键点击（或长按）喇叭按钮来选择文件
- 上传的图片保存在浏览器localStorage
- 刷新页面后仍然保留

### 默认外观
- 无图片时显示🔊表情符号
- 加载失败时自动降级到表情符号
- 图片大小18x18像素

## 使用示例

### 播放读音
```
用户在识词或拼写页面看到单词时，点击喇叭按钮🔊
→ 系统调用speakWord()函数
→ Web Speech API播放英文读音
```

### 上传自定义喇叭图片
```
用户右键点击喇叭按钮
→ 文件选择对话框打开
→ 用户选择PNG图片
→ 图片转换为Base64并保存到localStorage
→ 所有喇叭按钮的图标更新为新图片
```
